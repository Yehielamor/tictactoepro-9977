# tictactoepro-9977

TicTacToePro is an advanced, real-time multiplayer Tic Tac Toe game platform that enhances the classic experience with modern features. It allows users to play against friends or AI, track statistics, and compete in tournaments, all within a sleek, responsive interface. The platform focuses on quick matchmaking, interactive gameplay, and social engagement to make Tic Tac Toe exciting and competitive for players of all skill levels.

## Tech Stack
React (Vite) + Tailwind CSS + Supabase

## Setup
Run `docker-compose up` to start the infrastructure.
